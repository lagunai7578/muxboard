# Changelog

## 1.0.0

First release-shaped cut of muxboard.

- tmux-first command center for panes, agents, and other long-running terminal work.
- Fleet and Details view with stable columns, pane heat, live state, and explicit send-list flags.
- Fast output and tmux focus flows that work locally and over SSH-safe tmux setups.
- Agent-family lanes for Codex, Claude Code, Opencode, Aider, Gemini, and generic agent panes.
- Multi-pane send-list control, named fleets, review-before-send broadcast preview, and lane sends.
- One-key next moves for common waiting prompts, plus deeper fleet actions behind the More menu.
- Local desktop and bell notifications with SSH-safe fallback behavior.
- XDG-style config and state, with persisted muted alerts, recent commands, macros, and saved fleets.
- GitHub-first install path, package verification, and tag-driven Linux/macOS release artifacts.
- Unit coverage plus live tmux e2e coverage for search, send lists, review send, tmux focus, groups, rebinding, and alert-muting flows.
